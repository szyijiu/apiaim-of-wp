# Tokens分发平台首页规范

## 项目概述
- 项目名称: NexusAI Tokens分发平台
- 类型: 单页营销网站
- 核心功能: 展示AI tokens分发服务
- 目标用户: 开发者、AI企业、创业者

## UI/UX规范

### 布局结构
- 全屏首屏(100vh)带粒子星空背景
- 导航栏固定顶部
- 主内容区: 核心数据展示 + 功能卡片
- 底部: 合作伙伴/联系信息

### 视觉设计

**色彩系统**
- 主背景: #0a0a0f (深邃夜空)
- 主色调: #00f0ff (赛博青)
- 辅助色: #8b5cf6 (神秘紫)
- 强调色: #ff00aa (霓虹粉)
- 文字主色: #ffffff
- 文字次色: #94a3b8
- 渐变: linear-gradient(135deg, #00f0ff 0%, #8b5cf6 50%, #ff00aa 100%)

**字体**
- 标题: "Orbitron", sans-serif (科幻风格)
- 正文: "Rajdhani", "Microsoft YaHei", sans-serif

**视觉效果**
- 粒子星空动画背景
- 发光边框效果 (box-shadow: 0 0 20px rgba(0,240,255,0.5))
- 玻璃态卡片 (backdrop-filter: blur(10px))
- 扫描线动画效果
- 文字发光效果

### 组件
1. 导航栏: logo + 导航链接 + 登录/注册按钮
2. 英雄区: 主标题 + 副标题 + 动态数字计数器 + CTA按钮
3. 数据展示区: 3个发光卡片显示核心数据
4. 功能卡片区: 6个功能特性卡片
5. 背景: 动态粒子系统

## 功能规范
- 粒子背景自动动画
- 数字滚动动画
- 卡片悬停发光效果
- 平滑滚动导航
- 响应式适配
